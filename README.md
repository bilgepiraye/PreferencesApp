Project Overview
This project is a multi-screen React Native application demonstrating real-world state management patterns as required in Lab 6.

Key Features:
Authentication: Login/Logout flow with persistent session state (Context API + AsyncStorage).
Theme Preference: Dark/Light mode toggle that persists after app restart (Custom Hook).
State Management: Usage of `useState`, `useContext`, and `useEffect`.
Navigation: Stack navigation between Login, Home, and Settings screens.

Project Structure
* `/screens`: Contains Login, Home, and Settings screens.
* `/context`: Contains AuthContext for global state management.
* `/hooks`: Contains useTheme custom hook for theme logic.
